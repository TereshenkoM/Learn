Размер всех баз данных в кластере:

```postgresql
select
    pg_database.datname,
    pg_size_pretty(pg_database_size(pg_database.datname)) as size
from pg_database;
```

Чтобы посмотреть размер всех таблиц в схеме с их индексами и toast-таблицами:

```postgresql
select
    tablename,
    pg_size_pretty(pg_total_relation_size(schemaname||'.'||tablename))
from pg_tables
where schemaname='public';
```

Размер индексов для таблиц схемы:

```postgresql
select 
    pg_size_pretty(pg_total_relation_size(pg_class.oid)) as index_size,
    pg_indexes.indexname
from pg_indexes
join pg_class on pg_class.relname = pg_indexes.indexname
where pg_indexes.schemaname = 'public';
```
