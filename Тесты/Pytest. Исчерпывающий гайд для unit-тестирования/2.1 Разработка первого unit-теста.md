
**Пример теста**
```python
# test_example.py
def find_elem_index(text: str, element: str) -> int:
    """Находит индекс первого вхождения element в text."""
    return text.index(element)

def test_find_elem_index():
    """Проверка функции find_elem_index"""
    assert find_elem_index("abcdef", "c") == 2
```

**Пояснение:**

- Функция `find_elem_index()` ищет индекс первого вхождения элемента в строку.
- Тест `test_find_elem_index()` проверяет корректность работы функции.
- `assert` — ключевое слово для проверки ожидаемых результатов тестов.  Если условие ложно, pytest автоматически сообщает об ошибке.
- **Наименование тестов начинать с test_.**

Запустим тест командой
```
pytest
```

**Пример ошибочного теста**
```python
# test_example.py
def find_elem_index(text: str, element: str) -> int:
    """Находит индекс первого вхождения element в text."""
    return text.index(element)

def test_find_elem_index():
    """Проверка функции find_elem_index"""
    assert find_elem_index("abcdef", "c") == 3 # <-- изменили на 3
```

Запустим тест командой
```
pytest
```
