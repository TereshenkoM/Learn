
### Какие основные функции и команды для потоков используются в Python?

Основные функции и команды для работы с потоками в Python включают создание потоков с помощью модуля `threading`, запуск потоков методом `start()`, ожидание завершения потока методом `join()` и проверку состояния потока методом `is_alive()`

```python
import threading import time 
# Определение функции, которая будет выполняться в потоке 
def print_numbers(): 
	for i in range(5): 
		print(i) 
		time.sleep(1) 
# Создание потока 
thread = threading.Thread(target=print_numbers) 
# Запуск потока 
thread.start() 
# Ожидание завершения потока 
thread.join() 
# Проверка состояния потока 
if thread.is_alive(): 
	print("Поток все еще выполняется")
else:
	print("Поток завершил выполнение")
```


### Каковы преимущества мультипроцессинга в Python?
Преимущества мультипроцессинга в Python включают возможность использования всех доступных ядер процессора для параллельной обработки данных, изоляцию между процессами для предотвращения конфликтов при доступе к ресурсам и возможность обхода ограничений GIL для эффективной работы с CPU-интенсивными задачами (обработка видео, обработка фото, работа с большими массивами данных).
Например, рассмотрим пример использования мультипроцессинга для распределенного вычисления суммы квадратов чисел:

```python
import multiprocessing

def square_sum(numbers):
    return sum(x * x for x in numbers)

if __name__ == '__main__':
    numbers = range(1, 1000001)
    pool = multiprocessing.Pool()
    result = pool.apply(square_sum, (numbers,))
    print(result)
```

### Каковы ограничения мультипроцессинга в Python, и какие стратегии могут быть использованы для их преодоления?

Ограничения мультипроцессинга в Python включают

- накладные расходы на создание и управление процессами
- сложности с синхронизацией доступа к общим ресурсам
- невозможность обмена сложными объектами напрямую между процессами из-за отсутствия поддержки shared memory.

﻿Для преодоления этих ограничений можно использовать асинхронные подходы, механизмы межпроцессного взаимодействия и распределенные системы.

### Какие способы обмена данными между процессами существуют в Python?

Способы обмена данными между процессами в мультипроцессинге в Python включают использование каналов (pipes), разделяемой памяти (shared memory) и сетевого взаимодействия.

Например, вот как можно использовать каналы для обмена данными между процессами:

```python
import multiprocessing

def worker(conn):
    data = conn.recv()
    print("Received:", data)
    conn.send("Message received!")

if __name__ == "__main__":
    parent_conn, child_conn = multiprocessing.Pipe()
    p = multiprocessing.Process(target=worker, args=(child_conn,))
    p.start()
    parent_conn.send("Hello from main process!")
    print(parent_conn.recv())
    p.join()
```


### Можете ли вы описать, как вы будете обрабатывать CORS в FastAPI?
FastAPI имеет встроенное промежуточное ПО для управления CORS, которое можно добавить в экземпляр приложения. Чтобы включить его, импортируйте `CORSMiddleware` из `fastapi.middleware.cors` и добавьте его в свое приложение FastAPI с помощью метода `.add_middleware()`. Вам необходимо указать такие параметры, как `allow_origins` (список разрешенных доменов и IP адресов), `allow_credentials` (поддерживаются ли файлы cookie), `allow_methods` (разрешенные HTTP методы),и (какие HTTP-заголовки разрешены). Вот пример:

```python
from fastapi import FastAPI
from fastapi.middleware.cors import CORSMiddleware

app = FastAPI()

origins = ["http://localhost:3000"]

app.add_middleware(
    CORSMiddleware,
    allow_origins=origins,
    allow_credentials=True,
    allow_methods=["*"],
    allow_headers=["*"],
)

@app.get("/")
async def main():
    return {"message": "Hello World"}
```
